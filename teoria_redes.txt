O QUE É INFORMÁTICA?
   ciência que se dedica ao tratamento da informação mediante o uso de computadores e demais dispositivos de processamento de dados.
   Chama-se generecamente de informática ao conjunto das ciências da informação.

O QUE É INFORMAÇÃO?
   É  o resultado do processamento, manipulação e organização de dados de tal forma que represente uma modificação (quantitativa ou qualitativa) no conhecimento do sistema (pessoa, animal ou máquina) que a recebe.


			HARDWARE X SOFTWARE
O HARDWARE é a parte física do computador 
	é aquilo que vc pode chutar (monitor, gabinete)


O SOFTWARE é a parte 
	exemplos de software (pycharm, linux, minecraft, windows)


O FIRMWARE é o conjunto de instruções operacionais programados diretamente no hardware de um equipamento eletrônico. é armazenado permanentemente num circuito integrado (chip) de mem´roia de hardware
qualquer software armazenado sob a forma de memória de leitura, ROM, EPROM, ou ainda EEPROM e memória flash, no momento da fabricação do componente.

Memória RAM? Random Access Memory (memória do acesso aleatória) Você consegue acessar sem precisar acessar as memórias ateriores que você quer

Memória ROM - read only memory (Memória apensa para leitura)
